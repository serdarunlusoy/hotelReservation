Uses case name      : Search

Participating actors: User, Registrar or Admin

Priority            :High

Entry Condition     : Initiator enters the site

Exit Condition      : Initiator picks a hotel of his/her choice.

Flow of Events      : 1. The initiator types for a search and clicks the search button.
                      2. System displays a list of hotels fitting the initator's search. If
                         no fitting hotels are found, a 'can't be found' message is
                         displayed.
                      3. The initator picks a hotel of his/her choice and is redirected to
                      the hotel's reservation page.

Objects             : user, registrar, admin, search page, search bar,
                      search button, search list, reservation page
