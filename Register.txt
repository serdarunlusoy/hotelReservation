Uses case name      : Register

Participating actors: User

Entry Condition     : User visits the "mainPage".

Exit Condition      : User successfully registers or initiates the "Hotel Registration" use case.

Flow of Events      : 1. User clicks on the "Register Button".
                      		2. System displays "Registration Page".
		      3.1.User selects the "Hotel Registration Option".
	              		4.1.System initiates the "Hotel Registration" use case.	
		      3.2.User selects "Normal Registration Option" instead of "Hotel Registration Option" at Step 3.1.
                      		4.2. System displays the "Normal Registration Form".
		      5.User fills the "Normal Registration Form" and clicks the "Complete Registration Button".
				6.System completes the registration and displays the message "Registration Successful".

Objects:"mainPage","Register Button","Registration Page","Hotel Registration Option","Normal Registration Option", "Normal Registration Form","Complete Registration Button"
