Uses case name      : Display Error

Participating actors: Admin, Hotel User, Registrar or User

Entry Condition     : Login initiates

Exit Condition      : The Admin, Hotel User, Registrar or User receives the error and closes the Error dialog.

Flow of Events      : 1. An error dialog is displayed to the Login initator.
                      2. Initiator acknowledges the dialog and closes the Error dialog.

Objects             : admin, hotelUser, registrar, user, error dialog
